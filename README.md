**Temperature Converter in C**
**Project Overview:**

The Temperature Converter is a menu-driven C program that allows users to convert temperature values between different units. The application provides a simple and interactive interface where users can choose the desired conversion type and instantly view the converted result.

This project is designed to demonstrate the fundamentals of C programming, including functions, loops, conditional statements, and user input/output operations.

**Objectives:**
Perform temperature conversions accurately.
Practice modular programming using functions.
Implement menu-driven applications in C.
Improve understanding of user interaction and control structures.

**Features:**
Celsius to Fahrenheit conversion
Fahrenheit to Celsius conversion
Celsius to Kelvin conversion
Kelvin to Celsius conversion
Menu-driven interface
Continuous execution until the user chooses to exit
Input validation through menu selection

**Technologies Used:**
Programming Language: C

IDE: Visual Studio Code (or any C IDE)
